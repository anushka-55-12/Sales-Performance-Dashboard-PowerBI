# 📊 Sales Performance Dashboard — Power BI

s## 🧾 Project Overview
This project presents an **interactive Sales Performance Dashboard** built using **Microsoft Power BI**, based on the Superstore dataset (2014–2017).
The dashboard analyzes **4,922 orders** to uncover key business insights, track performance trends, and support data-driven decision-making.

---

## 🎯 Key Business Insights

* 🌍 **West region** leads with **$0.71M in total sales**
* 👥 **Consumer segment** contributes **50.76% of total revenue**
* 📱 **Phones** is the top-performing sub-category
* 💻 **Technology category** dominates with **36.59% contribution**
* 📈 Sales show a **consistent upward trend** from 2015 to 2018
